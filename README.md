# PCA Ordination & Licensure Study

An offline-capable progressive web app for studying toward PCA licensure and
ordination exams — spaced-repetition review of Bible Content, Theology,
Sacraments, Church History, the Book of Church Order, and Hot Topics.

Adapted from the spaced-repetition engine and UI shell of the Duff Greek study
tool. Source study materials live in `source_materials/`.
